Даны массив и запросы суммы на отрезке.

Формат ввода:
- Первая строка: n q
- Вторая строка: n целых чисел
- Далее q строк: l r (1-индексация)

Для каждого запроса выведите сумму элементов на отрезке [l, r].

Формат вывода:
- q строк с ответами

Ограничения:
- Все входные данные помещаются в 64-битный знаковый тип.
