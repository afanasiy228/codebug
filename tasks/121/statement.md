# Суммы по подмаскам

Дано число `n` и массив `f` размера `2^n`. Для каждой маски `mask`
нужно посчитать сумму `f[sub]` по всем подмаскам `sub`, таких что
`sub` является подмножеством `mask` (в битовом смысле).

## Формат ввода
Первая строка: `n`.
Вторая строка: `2^n` целых чисел `f[0..2^n-1]`.

## Формат вывода
`2^n` чисел — ответы для всех масок в порядке от `0` до `2^n-1`.

Ограничения:
- 0 <= n <= 20
- |f[i]| <= 10^9
