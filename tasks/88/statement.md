# Пары не больше K

Дан массив из `n` целых чисел. Нужно посчитать количество пар индексов `(i < j)`,
для которых выполняется `a[i] + a[j] <= K`.

## Формат ввода
В первой строке заданы `n` и `K`.
Во второй строке задан массив `a` из `n` целых чисел.

## Формат вывода
Одно число — количество подходящих пар.

Ограничения:
- 1 <= n <= 200000
- -10^9 <= a[i] <= 10^9
- -10^9 <= K <= 10^9
- Ответ может не помещаться в 32-битный тип.
