# Кольцевой маршрут

Есть `n` точек на плоскости. Необходимо стартовать из точки `1`,
посетить все остальные точки ровно один раз и вернуться в точку `1`.
Стоимость перемещения между точками `i` и `j` равна манхэттенскому расстоянию
`|x_i - x_j| + |y_i - y_j|`.

Найдите минимальную возможную стоимость такого маршрута.

## Формат ввода
Первая строка: число `n`.
Далее `n` строк с координатами точек `x y`.

## Формат вывода
Одно число — минимальная стоимость маршрута.

Ограничения:
- 2 <= n <= 15
- |x|, |y| <= 1000
