# Красные вершины

Дано дерево из `n` вершин. Изначально все вершины белые.
Обрабатываются два типа запросов:

- `1 v` — переключить цвет вершины `v` (белая -> красная -> белая).
- `2 v k` — вывести количество красных вершин на расстоянии не более `k`
  от вершины `v`.

Расстояние — число рёбер на пути между вершинами.

## Формат ввода
Первая строка: `n` и `q`.
Далее `n-1` строк с рёбрами дерева.
Далее `q` строк с запросами.

## Формат вывода
Для каждого запроса типа `2` выведите ответ.

Ограничения:
- 2 <= n <= 200000
- 1 <= q <= 200000
- 0 <= k <= n
