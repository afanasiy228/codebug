# Максимум в окне

Дан массив из `n` чисел и окно длины `k`. Для каждого окна
`[i, i+k-1]` выведите максимальный элемент.

## Формат ввода
Первая строка: `n` и `k`.
Вторая строка: массив из `n` чисел.

## Формат вывода
`n-k+1` чисел — максимумы окон, в одну строку.

Ограничения:
- 1 <= n <= 200000
- 1 <= k <= n
- |a[i]| <= 10^6
