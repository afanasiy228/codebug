# Оптимизация переходов

Даны три массива длины `n`: `x`, `a`, `b`. Требуется вычислить динамику:

- `dp[1] = 0`
- `dp[i] = min(dp[j] + a[j] * x[i] + b[j])` для всех `1 <= j < i`

Массив `x` задан в неубывающем порядке.

## Формат ввода
Первая строка: `n`.
Вторая строка: массив `x`.
Третья строка: массив `a`.
Четвёртая строка: массив `b`.

## Формат вывода
`n` строк — значения `dp[1]..dp[n]`.

Ограничения:
- 1 <= n <= 200000
- |x[i]|, |a[i]|, |b[i]| <= 10^9
- `x` неубывающий
