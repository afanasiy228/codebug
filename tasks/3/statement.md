# Сумма подотрезка массива

Дан массив из `n` целых чисел.  
Заданы два индекса `l` и `r` (1 ≤ l ≤ r ≤ n).  
Найдите сумму элементов на отрезке `[l, r]`.

## Формат ввода
n
a1 a2 … an
l r

## Формат вывода
Одно число — сумма элементов.

Ограничения:
- Все входные данные помещаются в 64-битный знаковый тип.
