# Следующий больший

Для каждого элемента массива найдите первый элемент справа,
который строго больше текущего. Если такого нет, выведите -1.

## Формат ввода
Первая строка: `n`.
Вторая строка: массив из `n` чисел.

## Формат вывода
`n` чисел — ответы для всех позиций.

Ограничения:
- 1 <= n <= 200000
- |a[i]| <= 10^6
