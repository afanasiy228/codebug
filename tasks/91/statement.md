# Много сумм

Дан массив из `n` целых чисел и `q` запросов. Каждый запрос задаёт границы
отрезка `[l, r]`. Для каждого запроса выведите сумму элементов на этом отрезке.

## Формат ввода
Первая строка: `n` и `q`.
Вторая строка: массив `a` из `n` чисел.
Далее `q` строк с парами `l r` (1 <= l <= r <= n).

## Формат вывода
Для каждого запроса выведите сумму на отрезке `[l, r]`.

Ограничения:
- 1 <= n, q <= 200000
- |a[i]| <= 10^6
