# Сумма на отрезке

Дан массив из `n` целых чисел и `q` запросов.
Каждый запрос задаётся двумя числами `l` и `r` — границы отрезка.
Для каждого запроса требуется вывести сумму элементов массива на отрезке `[l, r]`.
Индексация начинается с нуля.

## Формат ввода
`n q`
`a0 a1 … a(n-1)`
`l1 r1`
`l2 r2`
…

## Формат вывода
Для каждого запроса выведите одно число — сумму элементов.
