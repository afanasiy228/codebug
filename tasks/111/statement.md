# Пути в DAG

Дан ориентированный ациклический граф (все рёбра идут из меньшей вершины в большую).
Посчитайте количество различных путей из вершины 1 в вершину n по модулю 1e9+7.

## Формат ввода
Первая строка: `n` и `m`.
Далее `m` строк с ребрами `u v` (`u < v`).

## Формат вывода
Одно число — количество путей.

Ограничения:
- 2 <= n <= 20000
- 1 <= m <= 60000
