# Сколько не больше

Дан массив из `n` чисел и `q` запросов. Для каждого числа `x` нужно вывести,
сколько элементов массива не превышают `x`.

## Формат ввода
Первая строка: `n` и `q`.
Вторая строка: массив `a`.
Далее `q` строк с числами `x`.

## Формат вывода
Для каждого запроса одно число.

Ограничения:
- 1 <= n, q <= 200000
- 0 <= a[i], x <= 10^9
