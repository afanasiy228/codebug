# Количество вхождений

Даны две строки: текст `s` и шаблон `p`. Нужно посчитать,
сколько раз `p` встречается в `s` как подстрока. Перекрытия разрешены.

## Формат ввода
Первая строка: `s`.
Вторая строка: `p`.

## Формат вывода
Одно число — количество вхождений.

Ограничения:
- 1 <= |s| <= 200000
- 1 <= |p| <= |s|
- Строки состоят из латинских букв.
