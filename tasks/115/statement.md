# Самое частое

Дан массив из `n` чисел. Найдите значение, которое встречается чаще всего.
Если таких несколько, выведите минимальное значение.

## Формат ввода
Первая строка: `n`.
Вторая строка: массив `a`.

## Формат вывода
Одно число — самое частое значение (минимальное при равенстве частот).

Ограничения:
- 1 <= n <= 200000
- 0 <= a[i] <= 10^9
