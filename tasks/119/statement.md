# Различные подстроки

Дана строка `s`. Нужно посчитать количество различных подстрок,
которые можно получить, выбирая непрерывные отрезки строки.

## Формат ввода
Одна строка `s`.

## Формат вывода
Одно число — количество различных подстрок.

Ограничения:
- 1 <= |s| <= 200000
- Строка состоит из латинских букв.
