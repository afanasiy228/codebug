# Сумма по значениям

Дан массив `a` и `q` запросов. Каждый запрос задаёт диапазон значений `[L, R]`.
Нужно вывести сумму всех элементов массива, значения которых лежат в этом диапазоне.

## Формат ввода
Первая строка: `n` и `q`.
Вторая строка: массив `a`.
Далее `q` строк с `L R`.

## Формат вывода
Для каждого запроса одно число.

Ограничения:
- 1 <= n, q <= 200000
- 0 <= a[i] <= 10^9
- 0 <= L <= R <= 10^9
