# Количество различных

Дан массив из `n` чисел. Нужно определить, сколько различных значений
в нём встречается.

## Формат ввода
Первая строка: `n`.
Вторая строка: массив из `n` чисел.

## Формат вывода
Одно число — количество различных значений.

Ограничения:
- 1 <= n <= 200000
- 0 <= a[i] <= 10^9
