# Сумма по координатам

Даны `n` обновлений вида `x v`, означающих добавление `v` к точке `x`.
Затем идут `q` запросов `[l, r]`. Для каждого запроса нужно вывести сумму
всех значений в точках с координатами от `l` до `r` включительно.

## Формат ввода
Первая строка: `n` и `q`.
Далее `n` строк: `x v`.
Далее `q` строк: `l r`.

## Формат вывода
Для каждого запроса одно число.

Ограничения:
- 1 <= n, q <= 200000
- 0 <= x, l, r <= 10^9
- -1000 <= v <= 1000
