# Инверсии в массиве

Дан массив из `n` целых чисел. Инверсией называется пара индексов `(i < j)`,
для которой `a[i] > a[j]`. Найдите количество инверсий.

## Формат ввода
Первая строка: `n`.
Вторая строка: `n` целых чисел.

## Формат вывода
Одно число — количество инверсий.

Ограничения:
- 1 <= n <= 200000
- -10^9 <= a[i] <= 10^9
- Ответ может не помещаться в 32-битный тип.
