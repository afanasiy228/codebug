# Суммы на путях

Дано дерево из `n` вершин. В каждой вершине хранится значение.
Обрабатываются два типа запросов:

- `1 u v` — вывести сумму значений на пути от `u` до `v`.
- `2 u x` — присвоить вершине `u` значение `x`.

## Формат ввода
Первая строка: `n` и `q`.
Вторая строка: значения вершин `1..n`.
Далее `n-1` строк с рёбрами дерева.
Далее `q` строк с запросами.

## Формат вывода
Для каждого запроса типа `1` выведите сумму на пути.

Ограничения:
- 2 <= n <= 200000
- 1 <= q <= 200000
- |value| <= 10^9
