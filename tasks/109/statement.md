# Первая позиция

Дан массив `a` и `q` запросов. Для каждого числа `x` нужно вывести
первую позицию (1-индексация), на которой `x` встречается в массиве,
либо -1, если такого значения нет.

## Формат ввода
Первая строка: `n` и `q`.
Вторая строка: массив `a`.
Далее `q` строк с запросами `x`.

## Формат вывода
Для каждого запроса одно число.

Ограничения:
- 1 <= n, q <= 200000
- 0 <= a[i], x <= 10^9
