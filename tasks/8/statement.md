Дан отсортированный по неубыванию массив и число x. Найдите позицию (1-индексация) последнего элемента, не превосходящего x.
Если такого элемента нет, выведите 0.

Формат ввода:
- Первая строка: n
- Вторая строка: n целых чисел
- Третья строка: x

Формат вывода:
- Индекс последнего элемента <= x или 0

Ограничения:
- Все входные данные помещаются в 64-битный знаковый тип.
