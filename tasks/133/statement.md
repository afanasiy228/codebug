# Сумма в бесконечной таблице

Дан массив целых чисел a длины n.
Рассмотрим бесконечную таблицу, где элемент в строке i и столбце j равен:

a[(i + j) mod n]

Для этой таблицы требуется обработать q запросов.
Каждый запрос задается четырьмя числами x1, y1, x2, y2 и означает сумму элементов в прямоугольнике с верхним левым углом (x1, y1) и нижним правым углом (x2, y2), включая границы.

Индексация строк и столбцов начинается с 1.

## Формат ввода
n q
a0 a1 … a(n-1)
далее q строк вида
x1 y1 x2 y2

## Формат вывода
Для каждого запроса выведите одно число — сумму элементов в соответствующем прямоугольнике.

Ограничения:
- 1 <= n <= 4000
- 1 <= q <= 200000
- 1 <= x1 <= x2 <= n
- 1 <= y1 <= y2 <= n
