# Динамическая сумма на отрезке

Дан массив из n целых чисел.
Необходимо обработать q запросов двух типов:

1 l r — вывести сумму элементов массива на отрезке [l, r]
2 i x — заменить a[i] на x

Индексация начинается с нуля.

## Формат ввода
n q
a0 a1 … a(n-1)
далее q строк с запросами

## Формат вывода
Для каждого запроса первого типа вывести одно число.

Ограничения:
- 1 <= n, q <= 200000
