# LCP запросы

Дана строка `s`. Для каждого запроса `(i, j)` нужно найти длину
наибольшего общего префикса двух суффиксов: `s[i..]` и `s[j..]`.

Суффикс `s[i..]` означает подстроку, начинающуюся в позиции `i`
(нумерация с 1).

## Формат ввода
Первая строка: строка `s`.
Вторая строка: число `q`.
Далее `q` строк с парами `i j`.

## Формат вывода
Для каждого запроса выведите длину общего префикса.

Ограничения:
- 1 <= |s| <= 200000
- 1 <= q <= 200000
