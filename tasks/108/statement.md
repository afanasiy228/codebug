# Покрытие отрезками

Даны `n` отрезков на числовой прямой и `q` запросов-точек.
Для каждой точки нужно вывести, сколько отрезков её покрывают.

## Формат ввода
Первая строка: `n` и `q`.
Далее `n` строк с `l r`.
Далее `q` строк с точками `x`.

## Формат вывода
Для каждой точки одно число.

Ограничения:
- 1 <= n, q <= 200000
- 0 <= l <= r <= 10^9
- 0 <= x <= 10^9
