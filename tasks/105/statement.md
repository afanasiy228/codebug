# Палиндромные запросы

Дана строка `s` и `q` запросов `[l, r]`. Для каждого запроса определите,
является ли подстрока `s[l..r]` палиндромом.

## Формат ввода
Первая строка: строка `s`.
Вторая строка: `q`.
Далее `q` строк: `l r`.

## Формат вывода
Для каждого запроса выведите `YES` или `NO`.

Ограничения:
- 1 <= |s| <= 200000
- 1 <= q <= 200000
