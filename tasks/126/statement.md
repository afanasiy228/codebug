# Минимум на префиксе

Дан массив целых чисел длины `n`.
Для каждого индекса `i` необходимо вывести минимальное значение среди элементов `a[0], a[1], …, a[i]`.

## Формат ввода
Сначала целое число `n`, затем `n` целых чисел массива.

## Формат вывода
`n` чисел — минимумы для каждого префикса массива.
